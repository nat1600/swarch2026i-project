# swarch2026i-project
Language learning platform that turns your favorite series and movies into an active vocabulary experience. Capture words from subtitles, build your personal dictionary, review with smart flashcards, and level up with games and challenge.


| Feature |
|---|
|  Browser extension — captures words from subtitles |
| Authentication — register, login, JWT sessions |
| Vocabulary CRUD — save words and phrases with context |
| Flashcards — spaced repetition (SM-2) + Anki export |
| Solo games — fill-in-the-blank, word match, typing |
| Streaks — daily study tracking |
| Weekly leaderboard — top 10 with auto reset |
